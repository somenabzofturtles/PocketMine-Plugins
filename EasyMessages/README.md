#EasyMessages

###Commands:
Main command: **easymessages**, **em**

|Sub-command|Description|
|-----------|-----------|
|help|Shows all the sub-commands for EasyMessages|
|message|Sends a message|
|popup|Sends a popup|
|tip|Sends a tip|

###Permissions:
|Node|Default|
|----|:-------:|
|easymessages.command.easymessages|op|
